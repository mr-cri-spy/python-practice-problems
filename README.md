# Python Programming Problems 

This repository contains beginner-friendly Python programming problems and their solutions.

It is designed for students who are learning Python and want to practice problem-solving using:

- Conditional statements
- Loops
- Functions
- Basic algorithms

##  Problems Included

1. Salary Tax Calculator
2. Even or Odd Number
3. Largest of Three Numbers
4. Simple Calculator
5. Number Guessing Program
6. Multiplication Table Generator
